# 2021-sohu-textmatch
2021搜狐校园文本匹配算法大赛

# 建模思路
加载 bert 预训练模型，取最后一层序列输出的平均值作为句子对的embedding；任务类型embedding化与句子对的embedding相加，作为input输入到全连接层分类器。这样就可以达到用一个模型同时预测taskA和taskB的目标。
